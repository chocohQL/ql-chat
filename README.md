# QL-CHAT

## 项目介绍

QL-CHAT 是一个基于 SpringBoot+WebSocket+Sa-Token 搭建的仿微信实时聊天系统。

对应的仿微信桌面客户端：[https://github.com/chocohQL/ql-chat-desktop](https://github.com/chocohQL/ql-chat-desktop)
